# Dress Up Doll
## English
A dress up game made in godot that allows for very flexible and customizable expirience

### Main features include: 
- Color change for every item and every part of it
- Layering of items and ability to reorder them
- Pose changing, separate for every limb

### Roadmap 
- More item options
- More poses
